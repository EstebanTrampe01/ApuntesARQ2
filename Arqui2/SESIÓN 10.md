## IoT y Gemelos Digitales
- IoT conecta dispositivos del mundo real a la red
- Los Gemelos Digitales crean representaciones virtuales de esos dispositivos, reflejando estados en tiemp real

### Gemelo Dígital
Es una representación  virutal de un sistema o objeto físico
- Se actualizan constantemente con datos en tiempo real, por eso se diferencia de las simulaciones
- permite analizar y monitorear sistemas físicos

### Funcionamiento
recopilan datos de sensores IoT, se analizan en una plataforma digital, obteniendo información en tiempo real sobre estados y funcionamiento de sus procesos

## Tipos de Gemelos Digitales
### Component Twins
Piezas individuales, analizando partes criticas de un sistema como engranajes

### Asset Twins
Producto Completo, ayudan a comprender la interacción de distintas partes de un sistema para optimizar rendimiento

### System Twins
Múltiples productos o sistemas, interactuando en lineas de producción

### Process Twins
procesos complejos que involucran varios sistemas.
permiten una visión integral del rendimiento de fábricas completas
