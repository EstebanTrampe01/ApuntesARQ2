# Sistemas Automatizados 

**automática**: métodos y procedimientos para remplazar al usuario por cosas programadas.
**automatización**: es aplicar la automática en sistemas y procesos con mínima o ninguna intervención del usuario. (necesita de Sistemas y Procesos o Máquinas)

***Sistema***: nombre a lo que esta propenso a automatizar. Siendo un conjunto de objetos compuestos relacionados

***Procesos***: serie de actividades realizadas de forma ordenada

***Máquina***: diseña para cumplir un único propósito

## Proceso Industrial:

***Proceso Industrial***:  proceso donde algo entra - se transforma - se convierte en un producto

- Entradas: el material
- Transformación: la operación que cambia lo entrado.
- Perturbaciones: factores externos que afectan el producto
- Salida: producto final

**procesos continuos**
- Operan sin detenerse (24/7)
- Producción de Electricidad/ Petróleo / Papel

**procesos discretos**
- producen unidades individuales
- Ensamblaje de Celulares / autos 

**procesos batch (por lotes)**
- trabajan por lotes o tandas
- fabricante de medicamentos / - procesamiento de alimentos
-  un lote se procesa se detiene luego el otro.

## Sistema Automatizado Industrial
uso de sistemas o elementos computarizados y electromecánicos, con tareas de producción.
de operadores humanos a tecnológicos.
- protege integridad física.
- Libera la mente
- Aumenta la producción
- Reduce Costos
- Obtener información de cada proceso

### Automatización Industrial (AI)
uso de elementos o sistemas de tecnología de información para control de máquinas o procesos industriales remplazando a humanos.

ADEMÁS ayuda a los operadores en los trabajos físicos del trabajo, disminuyen la necesidad sensorial y mental del humano.

1. **Instrumentación Industrial** 
son transmisores y sensores que transforman información física a información eléctrica.

2. **Sistemas de Control y supervisor**
***control*** al ciclo de una maquina o proceso y controlarse por lectura de instrumentos 
***Supervisión*** interfaces que da el sistema automatizado para interactuar con seres humanos.

3. **Sistema de transmisión y recolección de datos**
el sistema automatizado es la fuente de recolección, y como se maneja esto, también como se transmite a una forma de almacenamiento.

4. **Aplicaciones de software**
Se utilizan en los Microcontroladores que aparecen en el sistema de control, en interfaces del sistema de supervisor (los dos anteriores que vimos "2 y 3").

---
### Control Manual
hay mayor intervención humana, el control se realiza de forma consigna (instrucciones que se le han dado al operador para realizar una tarea).

### Control Automático
moderno en sistemas de automatización, el control de máquinas se realiza por sensores de control, percibiendo cambios sutiles del ambiente o entorno; permitiendo a la maquina hacer ajustes necesarios para poder compensar estos cambios

### Finalidad de la automatización
la automatización de una maquina o proceso productivo simple tiene finalidad de liberar de forma física y mental al hombre

**Operaciones del sistema automatizado**
los sistemas automatizados incluyen:
- **Medición** : medir cambios físicos en su entorno
- **Evaluación** : analizar la información obtenida para tomar una acción "Control" 
- **Control** : acción de las operaciones de medición y evaluación
